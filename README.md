 🏙️ Urban Frames — Card Grid & Photo Gallery

A visually rich and responsive **urban-themed web project** featuring a **3-card layout** and a **3×3 photo gallery grid**.
Built using **HTML5 + CSS3**, this project focuses on aesthetics, layout structure, and modern UI/UX design principles.

---

 🌟 Project Overview

**Urban Frames** is a creative front-end project inspired by city life — combining:

* Architecture 🏢
* Street Art 🎨
* Night Life 🌃

It presents content through:

* Interactive cards (stories)
* A structured gallery (visual grid)

---

 🚀 Key Features

 🎴 Card Grid Section

* 3 responsive cards (desktop row)
* Converts to **1 column on mobile**
* Smooth hover animations
* Icon + image + description layout

 🖼️ Photo Gallery (3×3 Grid)

* 9 high-quality images
* Responsive behavior:

  * Desktop → 3×3 grid
  * Tablet → 2 columns
  * Mobile → 1 column
* Hover zoom effect + overlay icon

 🎨 UI / Design Highlights

* Unique color palette (Terracotta + Teal + Sand)
* Gradient text heading
* Soft shadows & depth effects
* Clean typography using Google Fonts
* Minimal yet artistic layout

 ⚡ Performance & UX

* Lazy loading images (`loading="lazy"`)
* Smooth transitions & animations
* Fully responsive design
* Lightweight (no frameworks)

---

 🛠️ Technologies Used

* HTML5 — Page structure
* CSS3  — Layout, animations, responsiveness
* Font Awesome — Icons
* Google Fonts (Inter + Poppins) — Typography
* Picsum Photos API — Random  high-quality images
